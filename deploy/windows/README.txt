NestSMC — Bot Trading di Windows (mode notifikasi Telegram)
===========================================================

PERSIAPAN SEKALI:
1. Install Python: https://www.python.org/downloads/  (CENTANG "Add Python to PATH")
2. Di Telegram, buka bot kamu (@SimpleCuan_bot) -> tekan START.
3. Siapkan TOKEN bot dari @BotFather (kirim /token).

MENJALANKAN:
- DOBEL-KLIK  start-bot.bat
- Saat diminta, tempel TOKEN. chat_id diambil otomatis.
- Cek Telegram: harus muncul "NestSMC aktif".
- Biarkan jendela hitam TERBUKA (tutup = bot berhenti). PC harus tetap menyala.

Token tersimpan di nestsmc.env (JANGAN dibagikan). Lain kali tinggal dobel-klik lagi.
